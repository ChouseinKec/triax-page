# Pseudo Store

## `usePseudoStore()`

Manages pseudo-class states (:hover, :active etc).

**Methods:**

### `getPseudos()`

Returns all available pseudo-classes.  
**Default Pseudos:**  

- default  
- hover  
- active  

#### `getPseudo()`

Returns currently selected pseudo-class.

### `setPseudo(value)`

Sets current pseudo-class.  
**Throws:** Error if invalid pseudo value provided.
